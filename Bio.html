Past Employments

  • Post-doctoral fellow
    Robert Bosch Centre for Cyber-physical Systems (RBCCPS), IISc Bangalore
    Host: Prof. Shalabh Bhatnagar
    Areas: Simulation-based Optimization, Simulation in Smart-city and Industrial IoT Applications,

  • Research Associate, Ajit processor development team, IIT Bombay.
    Ajit is a 32-bit SPARC V8-based processor developed at IIT Bombay. (IITB press article, Interview )
    My main contribution was to port the Linux kernel to the Ajit platform. I've also contributed to a detailed C-based executable model of the processor by building timing models for the memory subsystem (including a 2-level split cache, MMU, TLB and main memory interface) and basic device models (timer, UART, interrupt controller). I also contributed in building an automated instruction-set validation suite for Ajit.
